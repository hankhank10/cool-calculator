Simple databases in flask

What it does?

Takes an input database of characters from Happy Days

Processes this data to determine whether the character is cool or not

Outputs the data (including whether they are cool or not) to another database

Steps:

1. Check that there is nothing in the output database by visiting the /show_output_database endpoint. You should get an error.

2. View what is in the input_database by visiting the /show_input_database endpoint

3. Run the processing command by visiting the /process_everything

4. Check what is in the output database by visiting the /show_output_database endpoint. You will see that our complex algo has determined who is cool and who is not.

Extra credit

1. Make the input database massive by adding an extra random 1,000,000 people to it by visiting the /create_lots_of_people endpoint.

2. Imagine all the people. It's easy if you try, by visiting the /show_input_database endpoint

3. Run the processing command again and see how long it takes at /process_everything

4. Check what is in the output database by visiting the /show_output_database endpoint.

Notes

If you accidentally delete or corrupt the input database you can recreate it with the /create_input_database endpoint
